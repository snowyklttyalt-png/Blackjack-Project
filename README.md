**程式介紹:**  


遊戲為鋪克牌中的其中之一: 二十一點，以賭虛擬金的方式執行，操作開始玩家有$1000，並選擇要賭多少，在開始執行遊戲。  
玩家能輸入兩個值，分別為HIT(抽)和STAND(停)  
當玩家選擇停止或是抽超過21點，就換莊家抽  
莊家抽直到值有17或超過才停止  
如果玩家超過莊家抽的點數，算贏  
並依開始賭的錢來計算結果  
最後回到起始點，除非玩家金錢值為0，結束遊戲  

---

**程式安裝及使用方式:**  

1.下載成ZIP  
2.解壓縮檔案  
3.用Visual Studio開啟  
4.執行程式
